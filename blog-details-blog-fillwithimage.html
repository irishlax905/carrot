Fill a Typeface with an Image or Pattern with Adobe Illustrator

  • Nov 15th, 2013
  • Posted By: Carrot Press
  • 7 Minute Read

Let’s talk swatches and clipping masks in Adobe Illustrator, the tools that give you the ability to put patterns and images inside paintable elements. The advantage of Illustrator’s swatches is the ability to create custom swatches full of patterns and unique content inside a shape. Custom swatches can be used to create a unique and personalized fill for any vector image that is live paint enabled, like the example above.


Gradients can also be selected and saved as a swatch for later use with the Paint Bucket tool by creating a gradient, and simply dragging the thumbnail preview for that gradient in the ‘Gradient’ toolbar window into the ‘Swatch’ toolbar window. This will allow you to add your custom made gradient to the selected Swatch library.




Filling shapes with gradients is convenient and fun, but if you want to create a shape with an image or pattern inside of it, you’ll need to know a few more steps to take.

Step One: Open a New Document



The first step needed for the creation of a custom usable swatch from an image is to create that pattern in a new document. So let's go ahead and open up a new document by selecting File > New… and choosing a size that is large enough to suffice for your pattern or image.

Design your pattern or place your image so that no blank or empty space is left. If you’re making a pattern, consider the spacing of elements so if the object you’re trying to fill is larger than your saved swatch dimensions, the pattern or swatch will repeat itself evenly.






Step Two: Save to Swatch Library



Once you are finished with the previous step, we want to select all of the contents of your pattern or image by choosing Select > All or Command + A. Then, with the Swatches window open, drag the selected content over to Swatches. You will now see Illustrator has created a swatch and thumbnail for the selection however, we are not out of the woods yet.




If the thumbnails are too small, choose Large Thumbnail View in the hamburger menu of the Swatches window.


Once you see the pattern or image inside Swatches, we’ll want to choose the Swatch Libraries Menu, located at the bottom left-hand corner of the window. At the very top you should see Save Swatches.... This will prompt a path destination for your new swatch. Make sure to save your swatch somewhere other than where Adobe has your swatches saved by default, as these are tricky to find in the system folders of the Illustrator program.









Step Three: Use Your Swatch



Return to your original shape that you’d like to fill with a custom swatch and make sure that it is paintable by selecting all contents of the shape or artwork and choosing Object > Live Paint > Make. With the Live Paint Bucket tool selected, you will notice that there is a red highlight of the paintable areas with the Live Paint mode enabled.






Now the fun part, open the Swatches window in the original document if it’s not open already, and choose the Swatch Libraries Menu once more. This time, instead of saving to a swatch library, we’ll choose Other Library…, and select the swatch you just saved.



A new swatch window will appear with your saved swatch contained inside. With the Live Paint Bucket tool still selected, simply click to fill the shape with the custom pattern. You’ve made a custom-filled shape, congrats!






Fill Text with an Image



So, let’s go over how to ‘color’ a typeface with an image. If you have a swatch library that you’d like to add to, or simply prefer using swatches to fill anything from text to vectorized shapes, then this method is for you. First, we’ll need to select everything, Select > All and create outlines of our typeface to configure it for painting by choosing Type > Create Outlines






Once that’s finished, we can now convert the text into a paintable object by following the steps above. Once it’s paintable with the Live Paint tool, we’ll need to refer to the previous steps again, and fill the typeface with the image-based swatch.






There you have it, imagery on a typeface from a swatch application. Important note: The placement of the image cannot be moved directly in the original document. You will need to edit the placement of the image in the swatch document to change the placement of the image on the typeface.



Don’t forget to save your work and we hope you enjoyed this tutorial - be sure to stay tuned for more helpful Illustrator tutorials by Carrot!
