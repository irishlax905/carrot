Maya 2020: What’s New

  • Dec 27th 2019
  • By Carrot Press
  • 8 Minute Read


Maya 2020 introduces a whole new realm of possibilities for production artists, designers and 3D animators alike. With a new CPU and GPU rendering selection, Maya 2020 gives way for faster rendering time with the ability to tackle higher complexity animations. Whether you’re into rigging, modeling, lighting, texturing or animation, there are over 60 new features to discover. In this article, Carrot explores what’s new with Maya 2020 so without any delay, let’s dive right in.


At a Glance


  • - Matrix-driven Transform and Operation Nodes

  • - Proximity or UV Pin Nodes

  • - GPU-Accelerated Proximity Wrap Deformer

  • - Remesh Tool

  • - Retopologize Tool

  • - Bifrost + Maya SpecialFX

  • - Arnold 6 + Maya

  • - Cached + Image Cached Playback

  • - Animation Bookmarks

  • - Ghosting Preview


Rigging: Say Goodbye to DAG clusters

Credit: Autodesk.com

Tired of creating DAG clusters so complex and confusing that your animation team can’t even work on what you’ve created? Well, say goodbye to the confusing clusters of nodes. With the new release, Autodesk introduces matrix-driven transform and operation nodes, removing the need to compress and decompress animatable channels, and allowing the next team to seamlessly move from channel to channel for animation, without the unnecessary confusion or time-consuming process of finding a node to animate.


Proximity and UV Pin Nodes: Pin a Node on the Donkey


Credit: Autodesk.com

Another great node introduced to Maya 2020 is the pin nodes via the proximity or UV fashions. If you’re looking to constrain a prop to a character or perhaps, create a restraint on a specific node, you can use the proximity or UV pin nodes to achieve this effect.


Credit: Autodesk.com

If you’re working with a figure, facial expressions, or any texture that needs to stay smooth pixel-free during rendering, AutoDesk has noticed this and took action. Introducing the new GPU-accelerated Proximity Wrap deformer, for use on proximity wraps that are prone to deforming during the rendering phase.


Modeling: Fun New Texture Tools


Ever needing to create another, slightly different texture on top of a texture that is already there? Well, now you can with the new, Remesh tool, you can now configure the mesh of a texture to place a different texture in a specified area, as seen in the image below.


Credit: Autodesk.com

Don’t apologize, because it’s Retopologize - a perfect companion to the Remesh tool. The Retopology tool makes it easy to clean up your mesh after defining what textures you’ve selected. It miraculously maintains the original mesh while creating a new topology of quads for the surface mesh, a new technology for the process of remeshing in Maya.

Credit: Autodesk.com

SpecialFX at it’s Finest with Maya + Bifrost


Featured at SIGGRAPH 2019, Bifrost for Maya 2019 into 2020 has made some killer updates, including 200 bug fixes directly from user feedback. With new and improved, friendlier user experiences, performance and stability improvements, VP toolkit updates and much more. Take a look at this Bifrost 2019 Showreel for some amazing special effects created with Maya + Bifrost.




Rendering: Versatile Render Times


Credit: Autodesk.com

With the release of Arnold 6 with Maya 2020, we now have the ability to toggle from GPU to CPU rendering with a single click. Another amazing new feature, rendering is now more exciting with the ability to add any object into a viewport to watch your rendering update as it completes - a completely revolutionary new feature in Maya 2020.


Animation: Easier Playback


Credit: Autodesk.com

With the new and improved cached playback, users can now experience much faster animation playback speeds, allowing a cached playback of previous similar playbacks, a smart innovation with Autodesk.


You can now add image planes to the cached playback guestbook, as well - making image planes easier to view with short rendering previews. Maya does this by storing the images within the cache along with the other nodes and elements of the animation.


Credit: Autodesk.com

Now with animation bookmarks you can save a specific section you are working on, and continue to render and preview it as you work, making the animation of nodes much easier and convenient. You can also view the notes you’ve created for each bookmark.


Credit: Autodesk.com

There are several other innovative new features to the animation section of Autodesk Maya 2020, including the new Ghosting preview as shown above, allowing an animator to see past and future movements in the timeline relative to its current XY position.
